# Laboratório ZTNA + Web3

Consulte [MANUAL.md](MANUAL.md) para executar o experimento.

Estrutura principal:

- `compose.yml`: OpenZiti quickstart oficial, preservado como snapshot;
- `compose.lab.yml`: APIs e host ZTNA;
- `blockchain/`: contrato AccessRegistry e scripts;
- `scripts/`: instalação, configuração e monitoramento;
- `tests/`: cenários de segurança e desempenho;
- `results/`: resultados gerados localmente.
